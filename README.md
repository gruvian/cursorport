# cursorport
GUI Tool that automates porting of converted cursor images with [win2xcur](https://github.com/quantum5/win2xcur) for Windows to Debian. 
## Running

```shell
git clone https://github.com/gruvian/cursorport.git
cd cursorport
python3 main.py
```

### Built in
Python, bash

### Prerequisites
python3, win2xcur
